# 简介

本包是一个通用兼容性扩展包，用于为**原版物品**添加**自定义 tag**，从而方便各类模组之间进行联动与兼容。

## 本包的设计目标是：

在不破坏原版行为、不修改原版参数的前提下，为原版物品提供统一、可扩展的标记体系。

## 设计原则

在使用或二次开发本包时，请务必遵守以下原则：

### 1. 始终保持包体为最新版

本包会随着游戏版本与兼容需求持续更新。

请始终使用**最新版本**。

### 2.不修改任何原版物品参数

**所有原版物品的原始行为应与未安装本包时完全一致。**

## 功能说明

为原版物品添加统一规范的自定义 tag。

提供稳定的“中间层”，供其他模组引用。

降低联动成本，避免各项目重复定义相同分类。

提高整合包与长期项目的可维护性。
